Web Technologies

HTML

HTML is Hyper Text Markup Language. What this does is it is a foundation language for creating web pages. It also contains image and other web content.

CSS

CSS is a computer program that has types of code. It is a type of language for code also. Similar as HTML.

JavaScript

JavaScript is a coding language that gives us code to do. It is an advanced program. Also similar as CSS and HTML.

Bootstrap

Bootstrap is a popular CSS framework for developing responsive and mobile-first websites. It provides pre-built components and utilities that make web development faster and easier. Bootstrap uses a grid system for layout and includes JavaScript plugins for interactive elements.
